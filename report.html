CA2 Report

This page explains what Bootstrap components and JavaScript features I used for my CA2 one-page portfolio website. The website is built using Bootstrap 5 and my own HTML/CSS/JavaScript.

1) Bootstrap components used
  • Navbar (fixed-top)
    Used a Bootstrap navbar at the top of the page so the user can jump to each section.
  • Nav pills
    Used Bootstrap nav-pills for the navigation links (Home, About, Education, etc.).
  • Grid system (rows/columns)
    Used Bootstrap rows and columns for the Home section layout and the Contact section layout.
  • Buttons
    Used Bootstrap buttons for navigation (View Projects) and opening the contact modal.
  • Modal
    Used a Bootstrap modal to show contact information (Quick Contact).
  • Forms
    Used Bootstrap form controls for the Contact form (email, phone, category, message).
  • Alert
    Used a Bootstrap alert to show a success message after submitting the contact form.
  • List group
    Used Bootstrap list-group for Work Experience and Achievements to keep a clean layout.
  • Tables
    Used a Bootstrap table for Education module information.
  • Ratio (responsive embed)
    Used Bootstrap ratio to embed a responsive Google Maps iframe.
2) Dynamic features (JavaScript)
  • ScrollSpy (active nav highlight)
    Used Bootstrap ScrollSpy in JavaScript so the navbar highlights the section the user is currently viewing.
  • Project filter dropdown (show/hide sections)
    Used JavaScript to show or hide project sections based on the dropdown selection.
  • Contact form success message
    When the form is submitted, JavaScript displays a success alert and clears the form fields.
